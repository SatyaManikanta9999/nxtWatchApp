# NxtWatch – Video Streaming Web Application

NxtWatch is a YouTube-inspired video streaming web application built using React.  
The application allows users to browse videos across different categories such as Trending, Gaming, and Saved Videos with authentication and protected routes.

This project demonstrates React concepts like routing, component architecture, state management, and API integration.

---

## Features

- User authentication with login functionality  
- Protected routes for authenticated users  
- Home page displaying video content  
- Trending videos section  
- Gaming videos section  
- Saved videos functionality  
- Video details page  
- Like, dislike, and save video options  
- Search functionality for videos  
- Dark mode and light mode toggle  
- Loading and failure state handling
